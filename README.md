# 自动 3D 装柜原型

这是一个纯前端原型，支持：
- 标准柜型选择并自动带出参数（首批：`20GP / 40GP / 40HC / 45HC`）
- 高级模式下自定义柜体尺寸/限重（兜底）
- 尺寸单位 `inch/mm` 切换（默认 `inch`，内部统一按 `mm` 计算）
- 多产品参数输入（长宽高、密度、件数）
- 按“高度几层(可选是否限定) + 宽度几段（可选是否限定）”自动生成托盘总体尺寸，及托盘上的产品数量
- 生成每个产品的对应托盘尺寸的 3D模型，不同产品用不同颜色
- 自动装柜（碰撞检测 + 边界检查 + 限重检查）
- 手动拖拽3D block装柜（碰撞/越界/限重提示并回退）
- 演示级中英双语切换（默认中文）

## 运行

1. 在项目目录启动静态服务（任选其一）：

```bash
python3 -m http.server 8080
# 或
npx serve .
```

2. 浏览器打开：

- http://localhost:8080

## 使用流程

1. 选择柜型模板（默认可选 `20GP / 40GP / 40HC / 45HC`），自动带出尺寸和限重。
2. 选择尺寸单位（默认 `inch`，可切换到 `mm`）。
3. 设置高度层数（默认两层）、宽度分段（默认两宽），录入多个产品参数。
4. 点击“生成每种产品托盘尺寸与3D色块”。
5. 主界面 2/3 区域是 3D 柜内视图（带半透明边界），1/3 区域显示生成托盘的 3D 模型。
6. 选择“自动装柜”或“自主拖拽”，点击“执行当前方式”。

  - `Phase 6`：载具模板扩展到卡车场景（复用装载逻辑）
